# 🛒 Order Management System (C++)

## 📌 Опис проєкту

Це консольна система управління замовленнями, написана на C++.
Програма дозволяє створювати, редагувати, зберігати та завантажувати замовлення з файлу.

Система імітує реальний процес оформлення замовлення: від додавання товарів до оплати та доставки.

---

## ⚙️ Можливості

* ➕ Додавання товарів
* ❌ Видалення товарів
* 🔄 Зміна кількості
* 📦 Вибір доставки (кур'єр / самовивіз)
* 💳 Вибір оплати (картка / готівка)
* 📊 Відображення статусу замовлення
* 🔵 Візуальний прогрес (кружечки)
* 💾 Збереження у файл
* 📂 Завантаження з файлу
* 🛍 Додавання товарів з каталогу

---

## 🧾 Статуси замовлення

| Статус     | Опис        |
| ---------- | ----------- |
| Pending    | Створено    |
| Processing | Зв'язуємось |
| Confirmed  | Прийнято    |
| Preparing  | Готується   |
| Delivering | В дорозі    |
| Paid       | Оплачено    |
| Cancelled  | Скасовано   |

---

## 🧠 Логіка роботи

Програма автоматично:

* змінює статус залежно від дій користувача
* обчислює загальну суму
* додає доставку при необхідності
